Start Using Sora

Commands Reference

Complete guide to all available commands

üì¶ Repository Management

sora repos [--visibility all|public|private]
List all your repositories with optional filtering
sora create <name> [--private] [--desc "description"]
Create a new repository
sora view <repo>
Show repository information
sora stats <repo>
Display detailed repository statistics
sora rename <old-name> <new-name>
Rename a repository
sora update <repo> [--desc "text"] [--private|--public]
Update repository settings
sora delete <repo> [--force]
Delete a repository

üìÑ File Operations

sora ls <repo> [--path <path>] [--files-only|--folders-only]
List repository contents
sora cat <repo> <file-path> [--raw]
Read file contents
sora push <repo> <local-path> [--target <remote-path>] [-m "message"]
Upload file or folder to repository
sora rm <repo> <path> [--force] [-m "message"]
Delete file or folder from repository
sora pull <repo> [--dest <folder>] [--branch <name>]
Download repository to local machine

üå≤ Branch Management

sora branches <repo>
List all branches in a repository

üîê Authentication

sora --token <TOKEN> --check-account
Verify your GitHub authentication
sora --set-token <TOKEN> [--scope user|system]
Save token as environment variable (Windows)
